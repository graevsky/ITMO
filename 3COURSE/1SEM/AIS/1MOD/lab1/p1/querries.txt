Простой запрос для поиска фактов:
owns(arthur, revolver).

Запрос с использованием переменной для поиска объектов с определенными характеристиками:
owns(arthur, What).

Запрос с использованием логических операторов:
location(arthur, valentine), (enemy(arthur, Enemy); friend(arthur, Friend)).

Запрос, который требует выполнения правил для получения результата:
in_danger(arthur).

Запрос для определения взаимных друзей:
mutual friends(arthur, john).

Запрос, который проверяет, является ли Артур богатым:
rich(arthur).


Запрос для проверки, вооружен ли Артур:
