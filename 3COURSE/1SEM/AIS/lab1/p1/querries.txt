Простой запрос для поиска фактов:
owns(arthur, revolver).
location(micah, strawberry).


Запрос с логическими операторами
friend(arthur, Person), location(arthur, Location), location(Person, Location).
person(Person), \+ enemy(arthur, Person).


Запрос с использованием переменной для поиска объектов с определенными характеристиками:
owns(arthur, What).
owns(Person, Weapon), item(Weapon), (Weapon = revolver; Weapon = rifle).


Запрос, который требует выполнения правил для получения результата:
in_danger(arthur).
mutual_friends(arthur, john).
armed(arthur).
